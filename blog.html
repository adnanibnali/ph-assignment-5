Question-1: What are the different ways to select an element in the DOM?

getElementById(''), getElementsByClassName(''), getElementsByTagName(''), querySelector(''), querySelectorAll(''), .parentNode are different ways to select an element in the DOM

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. With innerHTML, we can get and set the HTML contents inside the tag.
2. With innerText, we can get and set the text content inside the HTML tag.
3. With textContent, we can get and set the text content inside the HTML tag, even it is hidden. we can not do this with innerText.

Question-3: What is event delegation in the DOM?

Event delegation is something where we don't add eventListener to evry child. Instead we add eventListener to parrent and know in which child event is hapening with the help of event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is something where when an event happens, it is passed through it's parrent element or bubble up, till the root of the document.
